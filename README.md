Install — add to your home screen for a full-screen shortcut.

iPhone & iPad (Safari)<br>
Open this app’s URL in Safari.<br>
Tap Share □↗ → Add to Home Screen → Add.<br>
Launch it from the home screen icon. Delete an old icon first if it opens the wrong page.<br>
<br>
Android (Chrome)<br>
Open this site in Chrome.<br>
Tap ⋮ → Install app or Add to Home screen, then confirm.<br>
Samsung Internet: Menu → Add page to → Home screen.<br>

Get the latest version<br>
After we publish updates, fully close this app and open it again so your device loads the newest files (especially if something looks out of date).<br>
<br>
iPhone / iPad: Swipe up from the bottom edge and pause (or double-click the Home button). Find this app in the app switcher and swipe it up off the screen to close it. Then open it again from your Home Screen icon.<br>
<br>
Android: Tap the Recents button (square or swipe up and hold), then swipe this app away to close it. Re-open from your home screen or app list.<br>
